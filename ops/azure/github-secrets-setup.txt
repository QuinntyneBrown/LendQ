GitHub Secrets Required for deploy-staging.yml
===============================================

1. AZURE_CREDENTIALS
   - Service principal JSON for Azure login
   - Generate with:

     az ad sp create-for-rbac --name sp-lendq-deploy --role Contributor --scopes /subscriptions/4a1b5113-89f9-4d27-acfe-581493385536/resourceGroups/rg-lendq-staging --sdk-auth

   - Copy the entire JSON output and save it as the secret value

2. SWA_DEPLOYMENT_TOKEN_STAGING
   - Azure Static Web App deployment token
   - Generate with:

     az staticwebapp secrets list --name swa-lendq-staging --query "properties.apiKey" -o tsv

   - Copy the token string and save it as the secret value

How to add secrets in GitHub
----------------------------
1. Go to the repo on GitHub → Settings → Secrets and variables → Actions
2. Click "New repository secret"
3. Add each secret with the name and value from above
